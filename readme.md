# Studiovision's Events Admin.

### Requirements

- [Node](https://nodejs.org/en/)
- [Webpack](https://webpack.github.io/)
- [Sass](http://sass-lang.com/install)

#### Development Environment

Clone or Fork this repo, and then run:

```sh
$ cd site
$ npm install
$ bower install
# Run
$ npm start
```

### Build

```sh
$ npm run build
```
